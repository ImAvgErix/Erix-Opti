# Changelog

## v1.0

Initial release.

### Features
- **Auto Optimize** — one-click, backup-first optimization with 100+ hardware-conditional tweaks
- **Hardware detection** — live WMI snapshot (CPU, GPU, RAM with DIMM capacity, storage, motherboard, BIOS, network, monitors, USB)
- **Erix Gaming power plan** — custom plan created from Ultimate Performance template with desktop/laptop variants
- **Timer resolution** — global timer resolution requests enabled for low-latency gaming
- **AI removal** — Copilot, Recall, AI companion, search AI/highlights disabled
- **Dark mode** — system-wide dark theme applied via registry
- **Status detection** — live probes for every registry, service, power, and network tweak
- **Tweak catalog UI** — real-time status per tweak with Active/Pending/One-shot/Skipped indicators
- **Correct OS labeling** — Windows 11 IoT/LTSC SKUs no longer mislabeled as Windows 10
- **RAM reporting** — shows installed DIMM capacity (not just OS-visible memory)
- **Single-file deployment** — self-contained .exe with bundled runtime
